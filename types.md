# Fundamental machine-level types
- `bool`
- `char`, `wchar_t`
- `int`, `long long`
- `float`, `double`, `long double`
- `void`

# Derived machine-level types
- `int*` - pointers
- `char[]` - arrays
- `double&`, `vector<int>&&` - references

# Complex types
- `struct`
- `class`
- `enum`
- `enum class`
